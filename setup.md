# SETUP

1. Vérifiez si tout le setup est alimenté
2. Allumez les 3 TV
3. Allumez le PC
4. Ouvrir la session
   - **Mdp: 15987** de la session Win
5. Lancez le logiciel Assetto Corsa
   - Icon dans la barre des taches
6. Lancez le logiciel DBox
   - Icon dans la barre des taches
   - Clique sur Library
   - Clique sur Assetto Corsa
   - Clique sur Launch
7. Choisir le bon paramètre :
   - Mettre le temps en infini
   - Sélectionnez les vitesses auto ou non, etc
   - Sélectionnez le circuit
8. Lancez une session
9. A vous de jouez
