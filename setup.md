# SETUP

1. Vérifiez si tout le setup est alimenté
2. Allumez les 3 TV
3. Allumez le PC
4. Ouvrir la session
5. Lancez le logiciel DBox
6. Lancez le logiciel Assetto Corsa
7. Choisir le bon paramètre :
   - Mettre le temps en infini
   - Sélectionnez les vitesses auto ou non, etc
   - Sélectionnez le circuit
8. Lancez une session
   - Attendez genre 4 min depuis le déverrouillage de la session Win
9. A vous de jouez
